#Booking
Запуск проекту - npm i > npm run dev

Тут ви можете забронювати дату для вашої відпустки, дані зберігатимуться в localStorage. 
- Якщо у вас є два ренжі броні і між ними всього лише один день то його як стартовий день ви вибрати не можете.
- Якщо наступний день після дня який ви хочете обрати заброньовани то день який ви хочете обрати буде недоступний.
- Якщо ви обрали  21 число і хочете обрати 28 але в цьому діапазоні вже є заброньовані дати то відповідно ви не зможете обрати 28 число як кінцеву дату тому що діапазони не можуть пересікатись
- Для того щоб забронювати дату вам потрібно пройти всі 4 етапи вибору дат та часу -> після цього кнопка букінгу стане активною
- !!! Якщо на якомусь етапі вам потрібно повернутись назад просто скиньте обрані дати натиснувги на кнопку "Cancel"
